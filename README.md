jQuery.simpleImageSlider
========================

An extremely simple image slider for jQuery

Usage:
=========================
```javascript
   $.simpleImageSlider('#myImageTagID', { 
     images : ['1.jpg','2.png', '3.jpg'],
     imageDir:'./images/',
     delayInSeconds: 5
   });
```
